# 📌 Maison d'Ali

A modern, visually-focused web project built with HTML, CSS, and JavaScript.

---

## 🎯 Project Purpose
This project is designed to deliver a clean and modern web experience focused on visual content such as images and videos.  
The goal is to create a simple, smooth, and aesthetic interface for users.

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## 📸 Features
- Image and video presentation  
- Clean and modern UI design  
- Responsive layout (mobile-friendly)  
- Lightweight and fast performance
